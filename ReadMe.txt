After Importing the project into your Eclipse workspace you must:

01) Right click the project name
02) Select "Properties"
03) In the window that pops up select "Java Build Path" in the sidebar
04) Select the "Libraries" tab
05) Click the triangle next to "lwjgl.jar"
06) Click "Native library location"
07) Select "Edit" in the right side menu
08) Select "External Folder"
09) Navigate to your workspace and into the project folder
10) Nativate to "lib/lwjgl/natives/"
11) Select the folder corresponding to your Operating System
12) Click "Ok" the appropriate number of times to close all open windows

Your system should now be able to run the Application
